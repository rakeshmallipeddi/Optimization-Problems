Example 1: Production Planning

Problem Statement

A furniture company manufactures three types of products: chairs, tables, and desks. Each product requires time on three machines:

  • Cutting machine: Chairs need 2 hours, tables need 4 hours, desks need 3 hours
  • Assembly machine: Chairs need 3 hours, tables need 2 hours, desks need 5 hours
  • Finishing machine: Chairs need 1 hour, tables need 3 hours, desks need 2 hours

Available time per week: Cutting (100 hrs), Assembly (120 hrs), Finishing (80 hrs)

Profit per unit: Chairs ($50), Tables ($80), Desks ($100)

Decision Variables:

x₁ = number of chairs, x₂ = number of tables, x₃ = number of desks

Example 1: Formulate the Problem

Enter your formulation:

Format: ax1 + bx2 + cx3 (enter coefficients and variables)
Consider hours needed per product and total available time

Example 1: Complete Solution

Correct Formulation:

Objective Function:
Maximize Z = 50x₁ + 80x₂ + 100x₃
→ We want to maximize profit. Each chair earns $50, each table $80, and each desk $100.

Constraints:
2x₁ + 4x₂ + 3x₃ ≤ 100 (Cutting time)
→ Each chair needs 2 hours, table needs 4 hours, desk needs 3 hours on cutting machine. Total available time is 100 hours.

3x₁ + 2x₂ + 5x₃ ≤ 120 (Assembly time)
→ Each chair needs 3 hours, table needs 2 hours, desk needs 5 hours on assembly machine. Total available time is 120 hours.

x₁ + 3x₂ + 2x₃ ≤ 80 (Finishing time)
→ Each chair needs 1 hour, table needs 3 hours, desk needs 2 hours on finishing machine. Total available time is 80 hours.

x₁, x₂, x₃ ≥ 0 (Non-negativity)
→ Cannot produce negative quantities of products.

Optimal Solution:

x₁ = 10 chairs, x₂ = 15 tables, x₃ = 8 desks

Maximum Profit: Z = $2,500

Example 2: Complete Solution

Correct Formulation:

Objective Function:
Minimize Z = 2x₁ + 3x₂ + 2.5x₃ + 4x₄
→ We want to minimize total cost. Food A costs $2/unit, B costs $3/unit, C costs $2.50/unit, and D costs $4/unit.

Constraints:
400x₁ + 300x₂ + 500x₃ + 200x₄ ≥ 2000 (Calories)
→ Food A has 400 cal/unit, B has 300 cal/unit, C has 500 cal/unit, D has 200 cal/unit. Need at least 2000 calories total. Use ≥ because it's a minimum requirement.

20x₁ + 25x₂ + 15x₃ + 30x₄ ≥ 100 (Protein)
→ Food A has 20g protein/unit, B has 25g/unit, C has 15g/unit, D has 30g/unit. Need at least 100g total.

x₃ = 2.4 units of C, x₄ = 1 unit of D

Example 3: Investment Portfolio

Problem Statement

An investor has $100,000 to invest in three stocks:

  • Stock S₁: Expected return 8%, Risk level 3
  • Stock S₂: Expected return 12%, Risk level 5
  • Stock S₃: Expected return 15%, Risk level 8

Constraints:

  • Total investment must equal $100,000
  • Maximum 50% ($50,000) in any single stock
  • Average risk: (3x₁ + 5x₂ + 8x₃) / 100,000 ≤ 5.5

Decision Variables: x₁, x₂, x₃ = amount invested in S₁, S₂, S₃

Example 3: Formulate the Problem

Enter your formulation:

Use decimal form for percentages (8% = 0.08)
Multiply both sides by 100,000 to eliminate division

Example 3: Complete Solution

Correct Formulation:

Objective Function:
Maximize Z = 0.08x₁ + 0.12x₂ + 0.15x₃
→ We want to maximize expected return. S₁ returns 8% (0.08), S₂ returns 12% (0.12), S₃ returns 15% (0.15).

Constraints:
x₁ + x₂ + x₃ = 100,000 (Total investment)
→ The sum of all investments must equal exactly $100,000. Use = because we must invest all the money.

x₁ ≤ 50,000 (Max 50% in S₁)
→ Cannot invest more than $50,000 (50% of total) in Stock S₁ to diversify risk.

x₂ ≤ 50,000 (Max 50% in S₂)
→ Cannot invest more than $50,000 (50% of total) in Stock S₂ to diversify risk.

x₃ ≤ 50,000 (Max 50% in S₃)
→ Cannot invest more than $50,000 (50% of total) in Stock S₃ to diversify risk.

3x₁ + 5x₂ + 8x₃ ≤ 550,000 (Risk constraint)
→ Average risk = (3x₁ + 5x₂ + 8x₃)/100,000 ≤ 5.5. Multiplying both sides by 100,000 gives 3x₁ + 5x₂ + 8x₃ ≤ 550,000.

x₁, x₂, x₃ ≥ 0 (Non-negativity)
→ Cannot invest negative amounts.

Optimal Solution:

x₁ = $25,000 in S₁, x₂ = $50,000 in S₂, x₃ = $25,000 in S₃

Maximum Expected Return: Z = $11,750

Example 4: Transportation Problem

Problem Statement

A company can use 4 transportation modes to ship 600 units:

  • Truck: Cost $5/unit, capacity 200 units, time 2 days
  • Train: Cost $3/unit, capacity 300 units, time 4 days
  • Air: Cost $15/unit, capacity 100 units, time 1 day
  • Ship: Cost $2/unit, capacity 500 units, time 7 days

Constraints:

  • Must ship exactly 600 units total
  • At least 30% (180 units) must arrive within 2 days
  • Budget limit: $4,000

Decision Variables: x₁, x₂, x₃, x₄ = units by truck, train, air, ship

Example 4: Formulate the Problem

Enter your formulation:

Minimize total shipping time (days × units)
Only truck and air meet the 2-day requirement

Example 4: Complete Solution

Correct Formulation:

Objective Function:
Minimize Z = 2x₁ + 4x₂ + x₃ + 7x₄
→ We want to minimize total shipping time. Truck takes 2 days, train 4 days, air 1 day, ship 7 days per unit.

Constraints:
x₁ + x₂ + x₃ + x₄ = 600 (Total demand)
→ Must ship exactly 600 units total across all transportation modes. Use = because demand must be met exactly.

x₁ + x₃ ≥ 180 (30% in ≤2 days)
→ At least 180 units (30% of 600) must arrive within 2 days. Only truck (x₁) and air (x₃) meet this requirement.

5x₁ + 3x₂ + 15x₃ + 2x₄ ≤ 4000 (Budget)
→ Truck costs $5/unit, train $3/unit, air $15/unit, ship $2/unit. Total cost cannot exceed $4,000 budget.

x₁ ≤ 200, x₂ ≤ 300, x₃ ≤ 100, x₄ ≤ 500 (Capacity)
→ Each mode has limited capacity: truck (200), train (300), air (100), ship (500) units.

x₁, x₂, x₃, x₄ ≥ 0 (Non-negativity)
→ Cannot ship negative quantities.

Optimal Solution:

x₁ = 100 units by truck, x₂ = 300 units by train

x₃ = 100 units by air, x₄ = 100 units by ship

Minimum Total Time: Z = 2,600 unit-days

Objective Function:
Maximize Z = 400x₁ + 600x₂ + 500x₃ + 700x₄
