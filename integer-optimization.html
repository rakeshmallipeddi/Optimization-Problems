Example 1: Capital Budgeting

EASY Finance

Problem Statement

A company has a $500,000 budget to invest in projects. Each project must be either fully funded or not funded at all (binary decision).

Project Investment Required Expected NPV
A$150,000$200,000
B$200,000$300,000
C$180,000$250,000
D$120,000$150,000

Additional Constraint: If Project A is selected, then Project C must also be selected (dependency).

Decision Variables: x₁, x₂, x₃, x₄ ∈ {0,1} (1 = select project, 0 = don't select)

Objective: Maximize total NPV

Example 1: Formulate the Problem

Enter your formulation:

Maximize the total NPV from selected projects
Total investment cannot exceed $500,000
If x1=1, then x3 must equal 1. Can write as: x1 <= x3
All variables are binary (0 or 1)

Example 1: Complete Solution

Correct Formulation:

Objective Function:
Maximize Z = 200,000x₁ + 300,000x₂ + 250,000x₃ + 150,000x₄
→ Maximize total NPV. Each project contributes its NPV only if selected (xᵢ = 1).

Constraints:
150,000x₁ + 200,000x₂ + 180,000x₃ + 120,000x₄ ≤ 500,000
→ Total investment across selected projects cannot exceed the $500,000 budget.

x₁ ≤ x₃
→ Dependency constraint: If Project A is selected (x₁=1), then Project C must be selected (x₃=1). If x₁=0, x₃ can be 0 or 1.

x₁, x₂, x₃, x₄ ∈ {0, 1}
→ Binary variables: Each project is either selected (1) or not selected (0). Cannot partially fund a project.

Optimal Solution:

Binary Integer Programming (BIP) Problem

x₁ = 0 (Don't select A), x₂ = 1 (Select B)

x₃ = 1 (Select C), x₄ = 1 (Select D)

Maximum NPV: Z = $700,000

Total Investment: $500,000 (exactly at budget)

Example 2: Media Selection

EASY Marketing

Problem Statement

A company wants to advertise on different media platforms. Each platform requires a minimum purchase if used.

Platform Cost per Ad Reach per Ad Min Ads (if used) Max Ads
TV$5,000100,000310
Radio$1,00020,000515
Online$80015,000420

Budget: $80,000

Decision Variables:

  • x₁, x₂, x₃ = number of ads on TV, Radio, Online
  • y₁, y₂, y₃ ∈ {0,1} = 1 if platform is used, 0 otherwise

Objective: Maximize total reach

Example 2: Formulate the Problem

Enter your formulation:

Maximize total audience reach
If TV is used (y1=1), then x1 >= 3. Use: x1 >= 3y1
Link x1 to y1 with upper bound

Example 2: Complete Solution

Correct Formulation:

Objective Function:
Maximize Z = 100,000x₁ + 20,000x₂ + 15,000x₃
→ Maximize total audience reach across all platforms.

Constraints:
5,000x₁ + 1,000x₂ + 800x₃ ≤ 80,000
→ Total advertising cost cannot exceed $80,000 budget.

x₁ ≥ 3y₁, x₂ ≥ 5y₂, x₃ ≥ 4y₃
→ Minimum purchase requirements: If platform is used (yᵢ=1), must buy minimum ads. If not used (yᵢ=0), xᵢ=0.

x₁ ≤ 10y₁, x₂ ≤ 15y₂, x₃ ≤ 20y₃
→ Links integer variables (ads) to binary variables (platform selection) and enforces maximum limits.

x₁, x₂, x₃ ≥ 0 and integer; y₁, y₂, y₃ ∈ {0, 1}
→ xᵢ must be non-negative integers (number of ads), yᵢ are binary (use platform or not).

Optimal Solution:

Mixed Integer Programming (MIP) Problem

x₁ = 10 TV ads, x₂ = 15 Radio ads, x₃ = 4 Online ads

y₁ = 1 (Use TV), y₂ = 1 (Use Radio), y₃ = 1 (Use Online)

Maximum Reach: Z = 1,360,000 people

Example 3: Facility Location

MEDIUM Supply Chain

Problem Statement

A company must decide which warehouses to open and how to assign customers to warehouses.

Warehouse Fixed Cost Capacity (units)
W1$100,000500
W2$150,000800
W3$120,000600

Transportation Costs (per unit):

Customer 1 (demand: 300) Customer 2 (demand: 400) Customer 3 (demand: 250)
W1$5$8$6
W2$7$4$5
W3$6$7$3

Decision Variables:

  • yᵢ ∈ {0,1} = 1 if warehouse i is opened
  • xᵢⱼ = units shipped from warehouse i to customer j

Example 3: Formulate the Problem

Enter your formulation (use x11, x12, x13 for W1, x21-x23 for W2, x31-x33 for W3):

Include fixed costs (for y variables) + transportation costs (for x variables)
Total shipments from W1 ≤ 500 only if warehouse is open (y1=1)

Example 3: Complete Solution

Correct Formulation:

Objective Function:
Minimize Z = 100,000y₁ + 150,000y₂ + 120,000y₃
    + 5x₁₁ + 8x₁₂ + 6x₁₃ + 7x₂₁ + 4x₂₂ + 5x₂₃ + 6x₃₁ + 7x₃₂ + 3x₃₃
→ Minimize total cost = fixed costs for opening warehouses + variable transportation costs.

Constraints:
Demand Constraints:
x₁₁ + x₂₁ + x₃₁ = 300 (Customer 1)
x₁₂ + x₂₂ + x₃₂ = 400 (Customer 2)
x₁₃ + x₂₃ + x₃₃ = 250 (Customer 3)
→ Each customer must receive exactly their demand from all warehouses combined.

Capacity Constraints:
x₁₁ + x₁₂ + x₁₃ ≤ 500y₁ (W1)
x₂₁ + x₂₂ + x₂₃ ≤ 800y₂ (W2)
x₃₁ + x₃₂ + x₃₃ ≤ 600y₃ (W3)
→ Can only ship from a warehouse if it's open (yᵢ=1). If open, total shipments cannot exceed capacity.

xᵢⱼ ≥ 0; y₁, y₂, y₃ ∈ {0, 1}
→ Shipment quantities are continuous non-negative; warehouse decisions are binary.

Optimal Solution:

Mixed Integer Programming (MIP) Problem

y₁ = 0 (Don't open W1), y₂ = 1 (Open W2), y₃ = 1 (Open W3)

Shipments:

W2 → Customer 1: 300, Customer 2: 400, Customer 3: 0

W3 → Customer 1: 0, Customer 2: 0, Customer 3: 250

Minimum Total Cost: Z = $275,850

Example 4: Portfolio with Transaction Lots

MEDIUM Finance

Problem Statement

An investor has $1,000,000 to invest in bonds. Bonds must be purchased in lots of $50,000 each.

Bond Return (%) Risk Score Min Lots Max Lots
A (Corporate)7%408
B (Municipal)5%2010
C (Treasury)4%1212
D (High-Yield)9%706

Constraints:

  • Must invest at least 80% of available funds
  • Average risk score cannot exceed 3.5
  • At least 20% must be in Treasury bonds (C)

Decision Variables: x₁, x₂, x₃, x₄ = number of lots purchased (integers)

Example 4: Formulate the Problem

Enter your formulation:

Each lot is $50,000. Return = lot_value × return_percentage
Total investment ≥ 800,000
Weighted average: (4x1 + 2x2 + x3 + 7x4)/(x1 + x2 + x3 + x4) ≤ 3.5

Example 4: Complete Solution

Correct Formulation:

Objective Function:
Maximize Z = 3,500x₁ + 2,500x₂ + 2,000x₃ + 4,500x₄
→ Each lot is $50,000. Returns: 7% × $50k = $3,500, 5% × $50k = $2,500, etc.

Constraints:
50,000(x₁ + x₂ + x₃ + x₄) ≥ 800,000
or: x₁ + x₂ + x₃ + x₄ ≥ 16
→ Must invest at least 80% of $1M = $800,000 = 16 lots.

4x₁ + 2x₂ + x₃ + 7x₄ ≤ 3.5(x₁ + x₂ + x₃ + x₄)
or: 0.5x₁ - 1.5x₂ - 2.5x₃ + 3.5x₄ ≤ 0
→ Average risk score cannot exceed 3.5. Multiply both sides by total lots to eliminate division.

x₃ ≥ 0.2(x₁ + x₂ + x₃ + x₄)
or: -0.2x₁ - 0.2x₂ + 0.8x₃ - 0.2x₄ ≥ 0
→ Treasury bonds must be at least 20% of total investment.

x₁ ≤ 8, x₂ ≤ 10, 2 ≤ x₃ ≤ 12, x₄ ≤ 6
→ Individual bond limits and minimum Treasury requirement.

x₁, x₂, x₃, x₄ ≥ 0 and integer
→ Must purchase whole lots only (integer variables).

Optimal Solution:

Integer Programming (IP) Problem

x₁ = 6 lots ($300k in Corporate), x₂ = 0 lots

x₃ = 10 lots ($500k in Treasury), x₄ = 4 lots ($200k in High-Yield)

Maximum Return: Z = $59,000

Example 5: Production Planning with Setup Costs

HARD Supply Chain

Problem Statement

A factory produces 3 products over 4 time periods. Each product incurs a setup cost if production occurs in that period.

Product Setup Cost Variable Cost/unit Holding Cost/unit/period Capacity/period
P1$5,000$20$2500 units
P2$8,000$30$3400 units
P3$6,000$25$2.5450 units

Demand by Period:

Period 1 Period 2 Period 3 Period 4
P1300400350300
P2200300250350
P3250300400300

Decision Variables:

  • xᵢₜ = units of product i produced in period t
  • Iᵢₜ = inventory of product i at end of period t
  • yᵢₜ ∈ {0,1} = 1 if product i is produced in period t

Example 5: Formulate the Problem

This is a complex problem. Focus on Period 1 for Product P1:

Minimize: Setup costs + Production costs + Holding costs
Ending inventory = Beginning inventory + Production - Demand. Initial inventory = 0.
Can only produce if setup occurs: x11 ≤ M × y11, where M is large number (e.g., 500)

Example 5: Complete Solution

Correct Formulation (Simplified):

Objective Function:
Minimize Z = Σ(Setup costs) + Σ(Production costs) + Σ(Holding costs)
= Σᵢ Σₜ [Setup_costᵢ × yᵢₜ + Var_costᵢ × xᵢₜ + Hold_costᵢ × Iᵢₜ]
→ Minimize total cost across all products and periods including fixed setup costs when production occurs.

Key Constraints (for each product i, period t):
1. Inventory Balance:
Iᵢ,ₜ₋₁ + xᵢₜ - Dᵢₜ = Iᵢₜ
→ Inventory flow: Previous inventory + Production - Demand = Ending inventory.

2. Setup-Production Linkage:
xᵢₜ ≤ Capacityᵢ × yᵢₜ
→ Can only produce if setup occurs (yᵢₜ=1). If no setup (yᵢₜ=0), then xᵢₜ=0.

3. Capacity Constraint:
xᵢₜ ≤ Capacityᵢ
→ Production in each period cannot exceed capacity limit.

xᵢₜ, Iᵢₜ ≥ 0; yᵢₜ ∈ {0, 1}
→ Production and inventory are non-negative continuous; setup decisions are binary.

Optimal Solution (Sample):

Mixed Integer Programming (MIP) - Lot-Sizing Problem

Period 1: Produce P1=700, P2=500, P3=550 (all setups = 1)

Period 2: Produce P1=0, P2=0, P3=0 (use inventory)

Period 3: Produce P1=650, P2=600, P3=700

Period 4: Use remaining inventory

Minimum Total Cost: Z ≈ $95,000
